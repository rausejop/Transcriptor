# TRL Technology Readiness Levels (UE Innovation)

This skill encompasses the definitions and requirements for different TRL levels as defined by CONFIANZA23.

## TRL Categories

- **REQ1.01 - TRL 3 SCRIPT**: CLI tasks using Typer with Type Hints. Initial script & functions to solve the desired problem.
- **REQ1.02 - TRL 3 RESTRICTED**: Standard library-only CLI tool. No pip install. Zero external dependencies. Use of TCL/TK or argparse.
- **REQ1.03 - TRL 4 GUI_TOOL**: Stand-alone local desktop application (CustomTkinter, Flet, NiceGUI). Alpha version tested internally.
- **REQ1.04 - TRL 5 SPA**: Single Web Page Application (HTML5, Tailwind CSS v4). Alpha version tested by outsiders.
- **REQ1.05 - TRL 6 MVP**: Open beta fully functional MVP Application. Tested by selected end-users.
- **REQ1.06 - TRL 7 WEB APP**: Fullstack architecture (Next.js, FastAPI, PostgreSQL). Securing with CSP, CORS, and Rate Limiting.
- **REQ1.07 - TRL 8 SAAS**: Cloud-native distributed architecture (Docker/K8s, Redis, Auth0). OWASP Top 10 2025 compliance.
- **REQ1.08 - TRL 9 LLM_CORE**: Custom LLM training/RAG-Orchestration (Ollama, LangChain, LlamaIndex, ChromaDB).
- **REQ1.09 - TRL 9 LLM_ADVANCED**: Advanced agentic workflows (LangGraph, Pinecone, Ragas, Observability).
